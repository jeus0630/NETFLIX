# Netflix Clone App

**Netflix Clone App**은 넷플릭스의 기능들을 클론한 프로젝트 입니다.  
단순 API 호출이 아닌 **Admin**, **Backend**, **Frontend**를 각각 구현함으로서 실제 넷플릭스와 같이 동작하도록 만들었습니다.
